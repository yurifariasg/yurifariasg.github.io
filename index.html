Side Projects

Voice Controller for Spotify

An Android Wear app that allows users use their voice to start media playback on their phones from an Android Wear device. The main motivation to create this app was to play songs when you have your hands dirty (ie. when eating pizza!). The app was created when Android Wear was released and there was no alternatives to the Google Play Music service and Spotify had no support for Wear devices. Nowadays the project was discontinued but you can still check out the code. Github

Jungle Chief ("Pilot Project")

This was a game started as a side school project with a couple of friends highly motivated to build games. The main idea behind this project is to allow one or two players to compete simultaneously on the same device. Players race and fight in a "Mario Kart"-like game to cross the finish line. The game was created using only OpenGL ES 1.1 and Box2D. We had plans to port to Unity or other cross-platform framework but unfortunately the project was discountinued before this happens. Github
